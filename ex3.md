## branch edit

- branch: 기존 내용을 유지한 채 새로운 내용을 추가하고 싶을 때 사용
- check out: 특정 브랜치로 돌아가고 싶을 때 사용
- check out in SourceTree: 브랜치 이름을 더블 클릭

## merge

- 헤드 브랜치에 변경사항이 없고, 병합 대상 브랜치가 헤드로부터 시작된 경우 아주 쉽게 병합 가능
